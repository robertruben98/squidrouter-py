# squidrouter-py

Python client for the Squid Router API (Axelar cross-chain swaps/bridge aggregator). Sync + async, Pydantic v2.

## Install

```bash
pip install squidrouter-py
```

The distribution is `squidrouter-py`; the import package is `squidrouter`.

## Quick start

```python
import squidrouter
```

See the [API reference](api.md) for the full surface.
